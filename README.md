# 🍽️ Cloud Kitchen Food Ordering System

A full-stack web application built using Python and Django that allows users to browse food items, search for dishes, add items to a cart, and place orders online.

## 🚀 Features

- 👤 User Registration & Login
- 🍽️ Food Menu
- 🔍 Food Search
- 🛒 Add to Cart
- ➕ Increase Quantity
- ➖ Decrease Quantity
- ❌ Remove Items from Cart
- 📦 Place Order
- 📜 Order History
- 📂 Food Categories
- 🖼️ Food Image Upload
- 👨‍💼 Admin Panel

## 🛠️ Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## 💡 Future Enhancements

- Order modification within a limited time
- Online payment integration
- Live order tracking
- Customer reviews and ratings
